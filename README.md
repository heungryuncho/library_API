## 3. 느낀점

그런데 클론 코딩의 프로젝트 환경이 제가 진행하는 환경과 달라 마주하게 된 이슈들이 있었는데  
이를 해결해보고 이해가 안 되는 부분들을 주석 처리하며 공부하면서 진행하였습니다.
